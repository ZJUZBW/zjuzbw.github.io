# 九环奥术施法者

## 法术大师

**玩法链接（必看）** ---[正义之怒施法者入门和进阶手册](https://github.com/ZJUZBW/Wotr-AofS)，装备位置和DC细节都在里面

**介绍**：

- 从序章开始就能全程发力的法师，可以跟**预言学派法师**和研究法师结合就了解，法术大师、研究法师和预言学派法师在玩法上各有侧重，但是殊途同归；
- 法术大师的强力点在于5次奥法契约（一个奥法契约等于一次全法师法表施法，可以都给最高环法术位实现高环法术位的绝对数量和灵活性领先），共通学派超魔可以配合极效超魔权杖实现法术强极效而不需要靠装备猴戏，法术聚能既可以常驻轮级BUFF（甚至对道途法术也有效果）也可以为高等魔化武器/高等解法/地狱烈焰射线提供极高的加成。
- 法师的DC成型期就是8级，这个时候有德莱文的帽子，而且智力刚好升到22，配合狐之狡黠，有28智力（9调整值）。

**职业**：20法术大师 / 16法术大师4博学士，分别适配不同道途。

**推荐道途**：（不按照强度排名，而是道途契合度排名；如果是按照组合强度排名，那么第三章是巫妖＞灵使＞天使＞恶魔≈御衡者；第五章则都是无敌）

1. **灵使**：超能力顺序：神话4卓越魔法，神话6灵动魔法、神话8惊世神力；记得道途对话多选善良选项，才能解锁自信之心系列法术。	
   1. 回合制骑乘爱乌的时候，选中爱乌头像可以单独控制爱乌，这样爱乌就可以用融熔球魔杖/<风暴领主的决意>的雷暴，攻击主角，这样就能给元素注能法袍充能了。【注：如果爱乌还要使用瞬发权杖，记得关闭自动结束回合】

2. **巫妖**：巫妖之力选择，推荐夺命魔法+魔法吞噬者（作为搭配也可以在博学士拿一个牧师5环法术抵抗）+生命凋零/自选
   1. 【夺命魔法】，每天使用3轮，无视法术免疫，同时被法术影响的生物会无法施法（具体无视免疫的程度弱于亡者血脉，敌人免疫震慑的话还是无法震慑敌人，基本上只能用于魅影杀手/诡影杀手等）。
   2. 【魔法吞噬者】，让骷髅伙伴用戏法打主角，未穿透主角的法术抗力，主角就能获得下一个法术瞬发。
   3. 【噬星之寒】，触发寒冷和负能量易伤+目盲。
   4. 【生命凋零】，~~搭配颂圣之语和3环的甄选召石（麻痹让敌人敏捷变成1，召石触发生命凋零造成1敏捷伤害，刚好达成COMBO--肉体属性归零就死亡；正好召石范围跟颂圣一致，而且环位低数量多），可以直接达成超低成本AOE秒杀的效果（原本颂圣秒杀需要高10CL，且敌人豁免检定失败；这套方式变成只要敌人不免疫麻痹，高5CL即可秒杀），同理群体冰牢/群体定身怪物+甄选召石也行~~。【注：增效颂圣之语虽然能造成伤害且触发生命凋零，但是这个方式是先造成敏捷伤害再麻痹，无法造成秒杀】
3. **御衡者**：凝视先出增加法术DC、施法者等级、攻击加值、借机攻击AB；御衡者一身强度主要都在后期，后期跟法术大师极其契合。
4. **恶魔**：魔相顺序-神话4魅魔相/爆燃魔相、神话5爆燃魔相/魅魔相+空奢魔相+任意高等魔相，神话6淫梦魔相；【记得第二章开始就用《咒法与召唤的二元性》刷出充足怒火卷轴，可以主角用可以多一次恶魔狂怒次数；配合七美德之戒，后期11次恶魔狂怒还是勉强够用的】
5. **天使**：天堂宝剑顺序（也可以参照天使圣使的顺序）：4铲奸除恶（法师学这个就是打爆发的，菜刀优先长明烈焰）、6长明烈焰、8迅捷流光；天使光环顺序：5不息明灯、7穿透魔光、9和煦阳光。天堂宝剑+力场之锋+初学者手套+强效增效魔法飞弹猛的很。
6. **诡计大师**：不是说法术大师玩诡计大师就不强，而是说发挥不出诡计大师完全体，玩诡计大师法师建议上研究法师博学士。

**专长加点推荐**：

- 法术专攻系列：法术专攻

- 解法专长：解法余波、暴力解法

- 法术专精系列（通用原则）：专精的法术一定是常用法术（魔法飞弹、火球术、解除魔法、高等解除魔法、闪电链），或者是专精后有质变的（颂圣之语、地狱烈焰射线、前期的共用抵抗能量---不过法术大师和研究法师不需要）。比如专精地狱烈焰射线，可以在13级的时候配合法术聚能放出19CL的地狱烈焰射线，这个就是质变（从基础13d6伤害骰变成45d6伤害骰）；比如专精颂圣之语，就可以扩大法术的打击范围，比如敌人14HD，你本来13CL，颂圣之语影响不了他，但是专精以后变成15CL＞14HD，就可以对他造成必定目盲（豁免成功仅仅目盲轮数减半）。

  > **比如**：1-4级专精魔法飞弹，5-6级专精火球术，7-8级魔法飞弹，10级任意，11级专精地狱烈焰射线/闪电链，12-18级一般就是地狱烈焰射线、高等解除魔法、颂圣之语之间选择，有颂圣之语就一直专精颂圣之语，没有的话就根据最终能否达到15CL和19CL这个坎来选择是否专精地狱烈焰射线。19CL以后基本上都是专精颂圣/高等解除魔法。

- 说一下延时超魔的作用，不仅可以让BUFF时间翻倍，更关键的是可以延时对敌人造成负面状态的时间，比如风暴束/狮子吼/(高等）咆哮术的1轮震慑变成2轮，神圣之惩的1轮目盲变成2轮，尖刺啸耳的1轮眩晕变成2轮（配合项链变成4轮），恐惧术的豁免成功1轮战栗变成2轮战栗，这样很多的时间短但控制效果强力的法术就会变得非常强力。所以说偏好延时不仅有加BUFF方面的意义，也有战斗方便的收益。

**介绍**：预言学派强力之处在于，首先预言学派效果上等于牧师的善良领域善良之触+崇高领域激励命令，然后自带1/2法师等级的先攻加成，20级学派大招为先攻检定自动骰20。相当于自带1.5个牧师领域+神话精通先攻，大招绝对先攻（先攻检定自动骰20）对于法师来说更是最强的能力。后期第一先攻，抬手就可以秒BOSS。强力支援能力+强大的施法能力+最高的先攻，这个就是预言学派的优势。
