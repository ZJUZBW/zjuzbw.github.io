# 血怒者

## 原怒者

**职业**：20原怒者

- 注：新版本有人会出现替换不了狂暴特技的BUG，不是100%触发。遇到BUG以后，找到可以操作主角或者队友的地方，开关狂暴以后再升级就正常了。

**介绍**：血怒者里唯一强力的变体就是原怒者，原怒者是野蛮人的完全形态（最新版本甚至更离谱），是游戏里最顶级的4环菜刀，从一级强到20级。新版本有第二血承以后，有共计10个血脉专长，且可以分别替换每条血脉的血脉能力，也就是说全换的话有20个狂暴之力，可以保留双血脉里最强的几个血脉能力，替换掉没用的获得狂暴特技。可以说不同血脉组合的原怒者有不同的强力之处。

- **原怒者和突变斗士职业对比**：
  - 缺点：在于8~10级区间因为没有精通重击【有DLC6的话，用法师三环磨锋砺刃就行；DLC6以后，5级不出异种武器斩矛的算亏模】、强度不如突变斗士，解决疲劳问题之前比较难受；
  - 优点：除此之外无论是机制还是数值上全期都要比突变斗士强，新版本有奋不顾身以后，12级之后的强度更是无可匹敌（即时制为此可以放弃斩矛和突刺，冲锋过去吃攻击还能瞬间反击造成大量伤害，甚至优先堆AC也不是不行）。

**开卡**：人类，19力14敏12体10智10感14魅，背景扒手，需要玩刚掌波绊摔的流派则可以选习武者背景，

**血承组合**：

- **初始血承**：为了保证流程强度，4级血承能力一般是必须替换为狂暴特技的；所以初始血承尽量选择4级血承能力比较菜的，然后1级血承尽量强力一些。比如天界、炼狱、元素、亡灵、奥术。
- **血承组合原则**：两个血承的核心专长原则上必须包括战斗反射和精通先攻；剩下就是根据血承能力和道途专长套路自行组合。

**技能加点**：运动（必出）、沟通、灵巧/隐匿/巧手、察觉

**道途推荐**：全道途契合，也非常适合佣兵来玩；**搭配恶魔道途也非常强力，跟剑圣一样是纯职业非常适合恶魔道途的职业**。

- **【恶魔道途】缴械重击+擒奴魔相**：重击→缴械→擒奴魔相额外攻击→重击，可以连锁；敌人被缴械以后，有可能用徒手攻击，因为没有精通徒手专长所以每次攻击会触发己方的借机攻击，瞬间暴毙。**这个玩法不怕护命，因为就算护命防御了重击额外伤害也可以缴械重击+擒奴魔相额外攻击**。

**后期面板举例**：

亡灵（第一血脉，4级、8级、12级替换）+蛇（第二血脉，8级和16级替换），灵使道途满BUFF情况下，无领域等额外加成；其他道途也可以达到这种效果。

常态战斗面板预计有100AB、100+AC、60全豁免、40先攻、150伤害这种级别。

- - 力量=24+10士气（把戒指也算进去了，以下士气加值双倍都是同理）+8增强+6体型+6亵渎+4天赋+2其他+4餐食+4变化自如+2炼金=70（30）
  - AB=20BAB-6猛力+30力调-2体型+4狂暴+1加速术+1其他（鞋子）+6表现（夺命姿态）+2表现（戒指）+18士气+2暴力呼召+1其他（恶魔愤恨）+3运气（恶魔之灾）+5武器+1环境（杰作之锤）+1武器专攻+2天赋（恶魔改造）+3神话（天命所归）=91
  - 剩下是嗅血斗篷2、破敌2、夹击包抄4、隐匿2、冲锋2、双食物再3、眼镜对抗异界的洞察加成1等。

**血脉推荐**：

- 蛇：4级可以加AC，满级共计+5AC；12级可以+5触及范围。
  - 核心专长：战斗反射
  - 套路专长：寓守于攻、绊摔；瞒天过海
  - 添头专长：健壮、闪电反射、战斗施法

- 炼狱：1级能力炽焰+炎爆，12级飞翼，16级地狱火冲锋可以增强冲锋伤害
  - 核心专长：战斗反射
  - 套路专长：缴械；瞒天过海、威逼
  - 添头专长：盲斗、钢铁意志

- 元素：1级元素打击可以一直成长，最后变成常驻2d6对应元素伤害，~~属于A类附伤~~。大招免疫重击和偷袭，免疫对应元素伤害。
  - 核心专长：精通先攻、武器专攻、*猛力攻击、顺势斩*；猛力攻击和顺势斩不可能等6级和9级再出，所以一般利用不上
  - 套路专长：无
  - 添头专长：闪避、顽石强韧、闪电反射

- 亡灵：1级冲锋造成1轮战栗、4级给武器幽冥附魔、16级每天一次变成虚体（目前是无限次，每次狂暴都可以变虚体，超级猛）；
  - 核心专长：无
  - 套路专长：威逼
  - 添头专长：顽强、闪避、坚韧、健壮、灵活移动、钢铁意志

- 精类：12级狂暴自带加速术，20级给万物破敌
  - 核心专长：战斗反射、精通先攻
  - 套路专长：威逼
  - 添头专长：闪避、灵活移动、闪电反射、精通闪电反射

- 天界：1级对抗邪恶异界生物造成1d6神圣伤害且让武器变成善良阵营，12级飞翼
  - 核心专长：精通先攻、武器专攻
  - 套路专长：无
  - 添头专长：闪避、钢铁意志、灵活移动、健壮

- 奥术：16级狂暴自带战法转换，这样就可以16原怒者1鳞拳3博学士（拿机会主义者和神能）；但是这样得学双持久，而且可能需要道途支持才能常驻神能。
  - 核心专长：战斗反射、精通先攻、奥术打击、*猛力攻击*；猛力攻击不可能等6级再出，所以一般利用不上
  - 套路专长：无
  - 添头专长：钢铁意志、战斗施法、扰乱施法

- 龙：4级的天防（到16级共计+4；跟蛇血叠加）、12级飞翼
  - 核心专长：精通先攻、*猛力攻击、顺势斩*；猛力攻击和顺势斩不可能等6级和9级再出，所以一般利用不上
  - 套路专长：威逼
  - 添头专长：盲斗、顽石强韧、健壮

- **先确定想要的能力，剩下就得根据血脉专长强度来选择了，血脉专长还能决定后续的加点方向**。

**狂暴之力**：

- **必选**：野兽图腾系列3个（12级之前出完高等野兽图腾），夺命姿态系列3个（4级出夺命姿态，16级出致命精准），**奋不顾身**（DLC6之后新增，受到攻击会反击，搭配战斗反射和常备不怠非常猛，12级优先出这个）
- **可选**（根据情况切换使用姿态）：猛力姿态+毁灭猛击，防御姿态+旋身闪避，勇猛姿态+激发凶猛
- **备选**：灵敏嗅觉、8级左右出强健体魄（免疫恶心和反胃；前期用来免疫敌人的魅影蛛网的反胃效果）

**神话加点**：

- **最新版本**：1无限狂暴、2猛力攻击（神话）、3第二血怒血承、4精通重击（神话）、5变化自如/常备不怠、6精通先攻（神话）、7大法师护甲、8神话盔甲专攻（中甲耐受）、9屹立不倒/神话冲锋/常备不怠、10神话盔甲专攻（中甲回避）/神话冲锋
- 大力哥神话冲锋的优先级不是很高，最后再出或者不出都行，只有对一些不能重击的BOSS有奇效（再搭配猛力姿态更好），毕竟这游戏后期能冲锋的机会不多。

**武器**：第一章大砍刀<恐惧进击>（有DLC6就是要么先出异种武器擅长-斩矛，要么拿新月战斧配法师三环磨锋砺刃）、第二章转新月战斧<碎虫者>，然后有精通重击以后转弯刃大刀/巨斧<坟中歌者> （圣教军模式不能开自动！，新版本改动了重击范围计算方式，现在坟中歌者不算很强了，中规中矩的武器；）或者转去拿斩矛。
