# 正义之怒主角BD前言

## 关于适合主角的职业

​	以下职业，纯职或搭配进阶职业就有很高的综合实力，且全流程平滑。至于RP（Role player），首先纯职业就是最大的RP，其次不要觉得强的就不RP、弱的才RP，不要觉得你的RP才是RP。【注意：从（高等）魔化武器的BUG修复以后，序章和第一章菜刀少1AB，第二章8级以后少3AB，以前常态AB刚好够用的职业，现在可能会变得不太够用了。所以神话1改学变化自如的玩法其实是对这个BUG修复的部分弥补】

**法系**：研究法师博学士=法术大师＞法师（无变体-预言学派），圣使，先知/探寻者（这个偏向菜刀，但是本质还是要配合天使道途来施法）；

**非骑乘菜刀（更适合即时制）**：僧兵（御衡者）、剑圣（诡计大师；也可以灵使和恶魔）、妖态化形师（巫妖）、原怒者（诡计大师和恶魔）、突变斗士、歌者；

**骑乘菜刀（更适合回合制）**：僧兵（御衡者）、奥法骑手、神眷猎人（崇高+荣耀/战争）、圣职杀手（动物+幸运/疯狂/联盟）、屠魔游侠；

## 主角道途和主角职业的适配

**注**：某些职业本身强度就很高，玩什么道途都行，比如剑圣法师之类的，但是搭配合适的道途在流程中和最终属性上会更强；道途之间本身就有差距，不是说你要玩法师/剑圣，然后选恶魔道途，结果玩不公平难度在第三章受苦说这个BD很菜。而是说你想玩恶魔道途，想要渡过弱势的第三章且后期成型后又强力的BD的时候，才选剑圣/法师。

**以神话3或神话4开始评价，没进道途之前则需要按照职业本身强度来算**。

T0表示职业和道途非常搭配，机制非常强，新手也可无脑通关。

T1表示职业和道途非常搭配，第三章左右成型，但是需要自己稍微会加一些BUFF之类。

T2表示职业和道途有一些搭配，第四章左右成型；成型以后强度也很高，最终强度为主线拿完BUFF和装备的最终强度可以爽玩不公平难度的DLC1。

有小数点则表示大成型期更晚，至于最终成型强度，T度并不能表示出来（比如巫妖道途物理菜刀的第五章强度，因为没有席拉的护卫友伴和其剧情掉落的杰作之锤，且不能吃伟岸雄姿和缩小术，相对于其他道途的物理菜刀是逊色不少的），但是反正都能轻松通关不公平难度了。

## 物理职业玩法总评

**物理系职业的输出方式**：整轮攻击（双持多打、双手、天生武器）、借机攻击、重击、骑乘冲锋（奋力冲刺+矛系武器合计第一次攻击3倍伤害）、猛扑、战技流（绊摔、擒抱压制、缴械）。每个职业根据其职业能力、法表、BAB和额外专长数量，有各自独特的输出模式组合，也同时会有各自的优缺点和强弱势期。

### **僧兵玩法解析**

- **优势**：出门就有马可以骑乘（对抗中体型或以下敌人+1AB），奖励专长可以直接出奋力冲刺，拿矛系武器就可以1级就可以3倍冲锋，6级有气力克敌机先，可以克敌+冲锋3倍伤害秒杀BOSS；3级气力额外攻击、6级武训武器可以吃两仪回风额外攻击次数加成，整轮攻击次数多（6级极限5打，满级不靠杰作之锤极限8打）；而且本身武僧感上防AC不低，每级有先攻成长，20级绝对先攻。回合制骑乘还能全程整轮攻击。
- **劣势**：自身单次攻击增伤不够高（基本上只有武器训练的加成，需要从本职业之外获得高额单次攻击伤害加成），难度高了后期即时制骑马反而可能是累赘。限定守序阵营，主角的话不能进诡计大师道途实现物理伤害极限【注：好像只要第三章神话任务的时候阵营跟诡计大师不符，需要做转阵营任务的时候，诡计大师就要神话9才锁阵营，所以主角僧兵是可以的。要求就是保持守序阵营然后神话任务之前转阵营到绝对中立，做完任务再转回守序即可】，主角诡计大师的队友僧兵，在大部分情况下的输出效率上可能更强也说不定。
- **综合优缺点以后，可以得出**
  - **输出模板**：最适合的输出模板就是双手武器，单次攻击伤害最高。配合重击借机流+精通终势斩极限放大伤害，一轮清屏。最终武器斩矛-<悲报传达者>可以每攻击5次多一次攻击。
  - **玩法**：前期第一轮骑马拿长矛冲锋打精锐敌人，然后开气海打整轮攻击 或者 切武器为大砍刀用强力顺势斩打群怪；5级有加速术、6级武训两仪回风以后，用大砍刀/新月战斧打整轮攻击（6级配合加速术一轮有5打了）；然后9级点异种武器擅长-斩矛，10级点出斩矛精通重击以后换双手最强模板武器-斩矛。
  - 回合制僧兵可以全程骑马爆杀，即时制在不需要冲锋的时候可以不骑马而是自己靠变巨术（比如6级以后），毕竟武僧本身就是AC不低的。
  - **适合的道途**：因为是守序阵营，所以只能选天使或者巫妖或者御衡者。天使和巫妖提供更高的常驻单次攻击伤害加成，可惜巫妖后期少杰作之锤少两次攻击次数而且不能伟岸雄姿（只能靠可怖外表）。僧兵本身也有足够的神话能力空余来出双持久。
  - **强弱势期**：一级就可以骑乘冲锋3倍伤害爆杀了，20级更是有绝对先攻。全程都是砍王捏。
  - **适合的队友**：领域人（守序、幸运、疯狂、联盟、崇高等）、歌者（提供面板伤害、AB、AC、额外重击倍率和猛扑）

### **妖态化形师玩法解析**

- **优势**：4级获得变形以后就是5打满BAB+满偷袭骰成长，整轮伤害最高。5级靠化形师狂暴、加速术、神话1深渊陷堕，6级就有极限8次攻击的整轮攻击，满级不靠杰作之锤是极限10次攻击次数。而且本身AC成长足够高。
- **劣势**：4级变形之前职业基本上没什么加成；天生武器武器骰烂、重击属性烂导致伤害成长在6级以后过于线性，单体攻击伤害上限低（像诡计大师上限都是100%暴击5~6倍，一刀重击随便800+的），没有人工武器的种种特效。偷袭伤害可能会被免疫（偷袭首先要求夹击；遇到元素生物、异怪之类的根本打不出偷袭；遇到有类似精通直觉闪避这种免疫夹击的敌人必须打措手不及才能造成偷袭）。只有第一天生武器（即化形师狂暴里左上角那个爪抓）才能吃祝福武器、圣战之刃的加成，剩下的天生武器不行。不是标准的重击借机流体系的组件，给队友的借机加成低，很难实现循环重击借机十几倍放大伤害。
- **综合优缺点以后，可以得出**
  - **输出模板**：没得选，只能是变形+天生武器。
  - **玩法**：4级之前点猛力攻击拿长矛，猛力攻击后面也可以防止遇到免疫精准伤害的敌人导致伤害过低（平时感觉伤害够可以关猛力）；4级以后全程变形伤害拉满。点出粉碎防御保证遇到有精通直觉闪避的敌人也能打出偷袭；后期配合<疾风扫叶护膝>，点出精通终势斩，实现天生武器的AOE伤害。如果有歌者提供猛扑，高额攻击次数配合神话冲锋就是关键时候的增伤手段。
  - **适合的道途**：天使和巫妖提供更高的单次攻击附伤，天使道途不合书成型慢但是后期提供伤害更高（适合当天使道途合书主角的队友）。巫妖道途成型快实现伤害提早质变，顺便攻击次数多还能更多次触发吸血鬼之刃的回血。妖态化形师本身也有足够的神话能力富余来在神话34出双持久常驻法师护甲、护盾术、死亡之力、吸血鬼之刃。
  - **强弱势期**：4级开始无敌爆杀，进入巫妖道途以后继续伤害质变，伤害上限在没有队友额外加成和附伤BUG的情况下也就是一刀150左右（算上附伤BUG大概250+），但是胜在对于极少数免疫重击和护命的敌人伤害不会下降很多（可惜免疫重击一般都有免疫偷袭）。
  - **适合的队友**：领域人（守序、疯狂、联盟、崇高等）、歌者（提供面板伤害、AB、AC、额外重击倍率和猛扑）

### **剑圣玩法解析**

- **优势**：
  - 奥法武器附魔，5级就有锐锋配合斩矛，前期就有30%重击率，配合小幸运/祈福，爆发极高。后期还可以自附魔光能，幽冥，公正，神圣，破敌等。
  - 有智上防、智上伤、智上借机次数、智上重击确认检定、智上先攻、智上AB（奥法精准）；属性利用率极高。
  - 19级完美反应先攻检定必定骰20，配合智上先攻实现绝对先攻。
  - 20级武器宗师+1重击倍率，还有4级完美打击和完美重击实现最高物理爆发。
  - 远见打击持续12秒，战前/第一轮 远见打击 + 第一轮/第二轮 空间斩，无论什么敌人都能轻松命中。
  - 1级送武器擅长+武器专攻，拿异种武器的话，相对于其他需要用异种武器的职业，等于1级送两个专长。
  - 16级魔战士反击，敌人防御式施法无论是否成功，都会引发剑圣的借机攻击。正义之怒敌人有非常多的施法者（甚至有的恶魔一轮三次施法），剑圣额外攻击次数事实上远远不止面板的部分，而且这次攻击还能触发常备不怠的加成。
  - 优秀的生存能力，智上防+护盾术+镜影术。
  - 差强人意的6环法表，可以自理高等魔化武器/魔化武器/牛之蛮力等，节省队友法术位。还能用战法转换补满BAB。因为先攻第一，就算是回合制也可以瞬发次元门（用超魔权杖）突脸打整轮攻击+措手不及。
  - 剑圣9级能以剑圣等级当成BAB来选择专长，9级可以精通重击，11级可以直接出两个精通精通重击专长，诡计大师搭配剑圣成型速度最快。
- **劣势**：中BAB导致攻击次数成长慢（得战法转换+16级魔战士反击以后才超越一般物理职业），5级不能出包抄，前期自带职业增伤不够高（只有奥法武器附魔增强，元素附魔被恶魔抗性抵消得靠灵魂之灾转力场，13级才有智上伤）。
- **综合优缺点以后，可以得出**
  - **输出模板**：最适合的输出模板就是双手武器，单次攻击伤害最高，搭配完美重击和变巨术，一刀重击秒人。最终武器斩矛-<悲报传达者>可以每攻击5次多一次攻击。
  - **玩法**：前期攻击次数少就用锐锋附魔+强力顺势斩+终势斩打AOE，然后整轮攻击配合重击借机流+精通终势斩极限放大伤害，一轮清屏。
  - **适合的道途**：诡计大师，高重击概率和重击倍率更容易触发借机+终势斩，诡计大师能力更是跟剑圣完美搭配。
  - **强弱势期**：前期能实现跟无变体战士一个级别的常态AB，但是能打极高爆发伤害而且生存更强也能自理buff，5级锐锋+队友加速术以后伤害有质变；第三章开始常态AB慢慢被其他菜刀超过（但是主角有很多额外加成，常态AB对于主角基本上够用），但是能提供打BOSS的最终手段（空间斩+远见打击），属于是不用动脑子的选择。
  - **适合的队友**：领域人（幸运、疯狂、守序、联盟、崇高等）、歌者（提供面板伤害、AB、AC、额外重击倍率和猛扑）

## 关于道途强度

**第三章道途强度排行（考虑队友部分）**：天使合书＞巫妖合书＞诡计大师（察觉2阶）＞灵使九环施法者＝巫妖菜刀＞其他组合（剩下要么用到灰色机制-比如恶魔啮咬绊摔无限连等-虽然上限也就那样，或者基本上等于白板就懒得比了）；注意这个合书强的前提是你纯职或者满CL，别整个1混血术士跟巫妖合书这种弱智操作。

**第四章道途强度排行（考虑队友部分）**：这个时候第三章的强力组合基本上都是横扫的，恶魔和御衡者也慢慢好起来了，灵使菜刀也有起色。

**第五章道途强度排行（考虑队友部分）**：6个前期道途组合厉害的话，这里基本上都是无敌的，转后期道途不是跟前期道途神话4567比，而是跟继续走前期道途神话8910对比。比如第五章本来御衡者就起飞，还转什么魔鬼金龙传奇就是血亏。传奇道途强度属于那种你想用多少BUG，是不是空降洗点摆拍，继承多少前期道途能力来决定。

**神话10新晋组合**：灵使转魔鬼+19法术大师1混血术士（奥术+亡灵/任意）

## 关于粉碎防御

- 【注】：自从粉碎防御触发方式修复以后（以前不需要命中一次，现在需要命中一次以后后续的攻击才能措手不及），这个专长对于大部分职业的优先级就更低了。
- **正义之怒里**：核心难度或者更低的，粉碎防御没有什么意义，最多就是有偷袭伤害的职业可以学，以便更好地触发造成偷袭；困难难度有一点点学粉碎防御的价值，至于**以下说法都是基于不公平难度！！！**。
- 正义之怒里7级开始高等隐形术，后期再配合共用心灵屏障基本上可以完全代替粉碎防御【注：新版本黑水敌人新增4级游荡者职业等级，导致高隐的作用缩小了，粉碎防御的高光时刻大概就是在黑水了吧】。只有前期能早出粉碎防御且不会拖慢其他专长成型节奏的职业，我才会推荐7~9级点出粉碎防御（比如远程射手7级点出粉碎防御搭配队友的打击士气，或者队伍有诗人的末日挽歌）；组队情况下，对于非杀手/战士等多专长的职业，强行出粉碎防御的收益并不是很高，因为出了粉碎防御必定导致其他专长成型速度大幅度降低 或者会导致职业本身特性没法发挥出来。

## 关于回合制和即时制的夹击判定

- 回合制下，夹击敌人的判定为：敌人在两个队友的**威胁范围**内。
- 即时制下，夹击敌人的判定为：敌人被两个近战队友**同时攻击（就是仇恨线在同一个人身上）**。

**基于以上条件，回合制下“抓准时机”的效果大部分可以被包抄替代，所以优先级降低。**

**补充说明**：目前版本，由于包抄有BUG，导致无论是回合制还是即时制，近战端包抄可以完全代替抓准时机，所以敏系菜刀可以不出抓准时机了，基本上只有大力哥和远程射手点抓准时机有一定的优先级（或者最佳替代办法就是大力哥自带独行战术/一人成军，然后自己出抓准时机）。链接：[【正义之怒】关于包抄、抓准时机和护命的机制](https://tieba.baidu.com/p/7862623142)

## 完美级菜刀的真正体现

因为大家对于菜刀上限的感知非常不同，所以对菜刀强度梯度的说法有争议。以MOD扩展（还原桌面内容而非MOD原创内容）以后，真正的滥强菜刀（基本上找不出任何缺点，流程强度无敌，后期强度无敌，先攻极高，AB和伤害极高，有动物伙伴骑乘或者自身AC极高，等效专长极多）来做上限，这样就知道原版各种职业的优缺点和相对位置了。

**菜刀职业评价有几个维度**：

1. **常态AB**：以无变体战士为基准线，高于这个数值则算合格。
2. **攻坚能力**：瞬间爆发的AB，能够100%造成措手不及或者使用接触攻击的为最优。
3. **输出模式**：高重击范围＞低重击范围，攻击次数多＞攻击次数少，攻击范围大＞攻击范围小，回合制骑乘＞回合制无视地形猛扑（死神天降）＞回合制猛扑/瞬发次元门＞回合制非骑乘，精通终势斩 ＞ 无终势斩，单次伤害高＞单次伤害低
4. **先攻**：先攻越高越有优势
5. **自BUFF能力**：减少队伍压力，也可以自己使用一些独有BUFF。
6. **生存能力**：即全豁免/免疫能力、【反射闪避】、审判官的【坚定】、AC（骑乘可以大部分情况下用坐骑AC视为本体AC）。
7. **额外专长**：影响成型速度和玩法上限。
8. 其他技能检定等优势

## 骑乘以后被借机攻击的对象

骑乘以后远离敌人的攻击范围时，骑者和坐骑**都会触发敌人的借机攻击**，**优先攻击坐骑，然后攻击骑者**。

骑术无双（DLC5新增神话能力）：骑乘时，骑者和坐骑都不会因为移动而受到借机攻击。这个能力比较适合即时制的骑乘玩法，优先级也很高。

特技（灵巧）：你可以使用特技技能穿过一个敌人的威胁范围而不引发藉机攻击。当你用这种方式移动时，你只能用半速移动。

- 检定DC = 敌人的CMD（战技防御），而敌人的CMD日常是极高的，如果是不公平难度，灵巧检定的DC后期将会随便到达80以上。

**如果敌人只有一次借机攻击次数**：

​	敌人**借机攻击坐骑**以后就没有借机攻击次数了，所以不会攻击骑者

- 如果坐骑开启特技（灵巧）：
  - 检定成功则坐骑免受借机攻击，因此敌人还有一次借机攻击次数，骑者会受到借机攻击（若骑者也开启灵巧检定，且成功的话也可以免受借机攻击）。
  - 检定失败则坐骑受到借机攻击。

**如果敌人有一次以上的借机攻击次数**：

1. 都不开启灵巧特技：战斗日志会显示借机攻击坐骑和骑者，敌人消耗两次借机攻击次数；但是最终只攻击一次，**攻击骑者**。
2. 骑者开启灵巧特技：

   - 检定成功：敌人只会攻击坐骑，只消耗一次借机攻击次数（坐骑也开启灵巧检定，且成功的话也可以免受借机攻击）。
   - 检定失败：与 1 情况相同
3. 坐骑开启灵巧特技：

   - 检定成功：敌人只会攻击骑者，只消耗一次借机攻击次数（骑者也开启灵巧检定，且成功的话也可以免受借机攻击）。

   - 检定失败：与 1 情况相同（如果骑者也开启灵巧检定，且成功的话也可以免受借机攻击）

**综上**：难度不高的情况下，骑乘的时候，骑者因为AC不高，堆高灵巧检定以后，开启灵巧特技可以有效减少所受借机攻击。而坐骑尽量不要开启灵巧特技（会降移速），防止借机攻击转移目标（除非你能保证骑者的灵巧检定必定成功）。【PS：隐形以后，若是敌人没有识破隐形的手段，则不会被借机攻击和夹击。所以骑者可以用高等隐形】
