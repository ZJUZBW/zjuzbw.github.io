BD收录原则

基于游戏原版无MOD（有MOD的情况会单独注明），困难难度的全流程BD（非空降+不洗点）。从头开始玩不公平难度的，自行修改加点。

先说一下aaathl(本人)对于的队友Build原则和期待吧

  1. 原职业强度够的话，尽量维持原职业；如果要转职的话，则尽量少兼职以实现基本玩法；尽量实现一个职业的理论强度上限。
  2. 一个职业的关键是专长点法和玩法——大家对于队友评价不一致，就是因为不了解队友原职业的强力玩法。
  3. Build强度尽量不基于各种“良性”BUG，但是也不排斥利用BUG（这个是无法避免的，但是兼职时不会为了更好的利用BUG而去兼职---比如兼职战斗祭祀拿火焰祝福，而是本身这个兼职就有不错收益---比如兼职1级血怒者搭配装备也有较高收益）。也就是说就算某些BUG修复了，对给出的大部分BD的影响也不会很大。
  4. 队友加点和玩法均按照困难难度+组队模式为前提，适用不公平难度的BD会单独注明。
  5. 不会为了特意堆一些特别高的摆拍数据（比如困难难度里毫无意义的面板100+AC）而加点或兼职，所有数据均基于困难难度足够用为前提，尽量提高队友的各方面强度（AB、AC、伤害、辅助能力、豁免、技能检定、法术CL--自己能常驻轮级法术且不容易被解除魔法、法术DC），尽量让一个队友可以负责更多的队内工作。
  6. 队友的BD强度不基于特定职业的辅助（包括特定的牧师领域），比如棕毛变化师、炼金术士等能“注入”的职业、荣耀领域/联盟领域的牧师等，以保证队友BD的通用性。
  7. 基于第6条的补充，设定一个通用强力队伍配置，一般每个强力队伍都要有一个九环神术职业+九环奥术职业，通用buff即为任意九环神术职业+任意九环奥术职业（不包括巫师）都拥有的BUFF，比如加速术、牛猫熊狐枭鹰系列、（高等）英雄气概、伟岸雄姿、预警、英雄祈神、真知术、防死结界、行动自如、祈祷术、荣光爆发、高等隐形术、秩序护盾、变巨术/缩小术、朦胧术、移位术、防护阵营等。

注：

  • 开拓者系列，我本人只玩即时制，也基本不切换游戏模式，所以下面的所有由 aaathl 所写BD都是在即时制使用的。在回合制不一定合适（也许有回合制特有BUG，也许BD不适合回合制，回合制里的强度基于脑补）。

  • 相对于即时制，回合制对于玩法的增强和削弱如下：

  • 敏系缩小菜刀的输出能力极大削弱，因为经常打不了整轮攻击，需要有猛扑之类的才能补救。

    长柄变巨菜刀和远程射手的强度不变。

    标准动作的特殊攻击增强，比如要害打击和顺势斩。

    施法者极大增强，尤其是自发施法者。

    骑乘极大增强，比如骑乘可以全程夹击敌人，触发夹击和偷袭更容易，用强力顺势斩甚至可以对每个敌人都是夹击；而且回合制骑乘可以全程整轮攻击。

关于回合制和即时制的夹击判定（关键）：

  • 回合制下，夹击敌人的判定为：敌人在两个队友的威胁范围内。
  • 即时制下，夹击敌人的判定为：敌人被两个近战队友同时攻击（就是仇恨线在同一个人身上）。

基于以上条件，回合制下“抓准时机”的效果大部分可以被包抄替代，所以优先级降低。

补充说明：目前版本，由于包抄有BUG，导致无论是回合制还是即时制，近战端包抄可以完全代替抓准时机，所以敏系菜刀可以不出抓准时机了，基本上只有大力哥和远程射手点抓准时机有一定的优先级。链接：【正义之怒】关于包抄、抓准时机和护命的机制

这一套的思路是极致堆AC和豁免，然后靠正义烙印提供辅助，本体后期靠兼职原怒者补充AB后（如果队伍里有歌者，就不兼这个，变15圣武士）也有部分输出。比较适合那种喜欢有一个非常坚挺的主T的玩法。

技能加点： 3灵巧，主沟通、使用魔法装置和察觉（沟通可以换其他的，比如世界）。

思路： 保证生存能力的前提下，速升11级圣武士出正义烙印用来打BOSS，然后兼活体和先知疯狂堆AC，最后来一级原怒者保证命中和伤害（这作给了非常多的狂暴装备，1级狂暴就可以最多加6~7AB）；保证BD基本强度的情况下，最大程度地保留圣武士等级。优化以后，不需要靠队友来给护盾术或者用护盾术卷轴了。在后期面板AC足够以后，把防御式战斗关了。

后期玩法两种：

  1. 转双手无甲，法师护甲靠巫师（前面带个智力头环就行，后面用极致精神头环），护盾术靠2级活体解剖师（基础2cl+<烈火织法者>1cl，用<低等延时超魔权杖>以后6分钟，这样就可以持久到24小时了——<烈火织法者>只有在加buff的时候切出来用，其实真到了后期也不缺这点AC）。武器可以选巨斧<坟中歌者>、精灵曲刃<灭活者>、巨镰<扫荡>、弯刃大刀<厄运>。
  2. 转剑盾无甲，拿重盾（希望使者不朽的爱）和单手武器，伤害主要靠附伤，可以拿<夜空闪>、<三元素反曲刀>之类的玩元素狂潮，也可以拿长剑<攻守平衡>，弯刀<破晓花吻（善良版）>。

两者的AC在没有荣耀领域的情况下差不多的，满级魅调大概是20+2【魅力书】+6【增强】+4【炼金】 = 32（11）。

  1. 双手无甲的AC为4【护盾术】+11【敏调】（魅力代替敏捷）+11【鳞甲之拳魅上防】，这里是26点AC。有【护卫友伴】以后，魅力上AC【盾牌加值】，此时AC为11【护卫友伴】+11【敏调】（魅力代替敏捷）+11【鳞甲之拳魅上防】，这里是33点AC。
  2. 剑盾无甲的AC为7【+5重盾】+11【敏调】（魅力代替敏捷）+5【盾牌增强】（魔化盾牌）+1【盾牌专攻】，这里是24点AC。有【护卫友伴】以后，魅力上AC【盾牌加值】，此时AC为11【护卫友伴】+11【敏调】（魅力代替敏捷）+5【盾牌增强】（魔化盾牌）+1【盾牌专攻】，这里是28点AC。

前者比后者多了2~5AC，如果算上荣耀领域的话，再多10AC，猛力攻击加成和力调加成更高；后者可以拿一些特效非常好的重盾<希望使者不朽的爱>。

思路：奥法暴徒的职业定位本身就是一个多面手，但是由于正义之怒里敌人普遍拥有法术抗力（需要投资法术穿透的专长），沃尔吉夫的初始专长是双持又不能兼职武僧嫖AC，奥法暴徒的专长数量也很少，导致很多人对这个职业的体验非常一般（毕竟用不了一代的经典BD——18奥法暴徒1僧1活体）。所以要玩好双持奥法暴徒，就要对专长有非常好的规划。

题外话：

关于圣使和先知

  1. 前期圣使远超先知（升环快+忠诚祝福特别强）；中期无论是领域扩表（尤其是奇迹领域都算主领域）还是领域能力（联盟、疯狂、荣耀、崇高、幸运、善良），圣使都是强于先知的；后期两者都很强力，主要是侧重点不同，先知更适合主角玩（配合天使道途，战斗+自然，这样自身属性非常完美），圣使无论是主角还是队友都很合适。
  2. 先知的好处在于他是自发施法，法术位多（圣使是4+1领域法术位，先知是6法术位），加上牧师每环好用的法术就那几个，只要选对法术，后期先知的施法灵活性反而大于圣使。缺点是可知法术拿的慢，11级之前都需要搭配队友来补充一些buff，比如科米丽雅可以补共用延缓毒发、共用抵抗能量这类的，比如索希尔补充圣战之刃。而且先知由于没有领域的辅助能力，更适合BUFF机+塑能系输出。
  3. 扩表能力，圣使和先知都差不多强。圣使强力领域侧重于辅助队友和攻坚（联盟、疯狂、荣耀、崇高、幸运、善良），而先知秘示域更倾向于加强自身某种属性。
  4. 先知是主魅力的，所以先知（生命秘示域）的导能数量比圣使多很多，加血能力更强。

定位： buff机+动物伙伴之友+超级奶妈，宗教检定工具人+打击士气工具人 或者 法术轰炸

思路：

说一下恶魔移植，第四章竞技场的医生可以进行恶魔器官移植手术，效果很强，主角要移植的话，就跟聂纽一起去，然后聂纽先移植，要不然可能只能一个人移植。

  • 按理来说，卷轴专家的玩法应该是卷轴扩表+无限法术位（卷轴就是法术位，而且卷轴DC和CL比自己用法术书释放的还高），天生就是最强的轰炸机和buff机。但是卷轴法师10级大招有BUG：使用自己制作的卷轴时，不能用自己的CL而是用卷轴CL。导致聂纽就算20级也不能靠卷轴常驻轮级法术【目前，双持久已经不能常驻卷轴和能力了】。由于快捷物品栏使用非常不方便，大家都不太喜欢用卷轴----这个安装MOD（Quick Cast）可以解决。因此聂纽有时候只会被当成少3个法师专长的幻术专精法师使用，玩起来那是异常难受啊。。。

如何堆DC？

  1. 法术基础的DC是10，加上法术的基础环级（比如闪光尘是2环，就是2）
  2. 施法属性调整值的加成，比如卷轴专家是智力施法，智力调整值会加到法术的DC上；所以法师一定是加智力的，智力越高越好。
  3. 装备和专长的加成
  4. BUFF的加成

附伤套娃的玩法思路

近战

  1. 附伤套娃--多重附伤流（BUG）基本玩法：A类附伤——一般在战斗日志里独立结算一行，B类附伤——可以被A类附伤再次触发一遍。
  2. 常见A类附伤为14个（'巨灵化身'×4，'火焰烙印'，'轰鸣击'，【神之敌】，【天使之击】，战斗祭祀的【流水】/【火焰】祝福，巫妖道途的'吸血鬼之刃'，<高尚裁决>的1d6神力伤害，天使的'光之墙'---不清楚到底是A类还是B类了，<圣兽之爪>的挥砍伤害），A类附伤要看伤害来源，如果是来自队友的，则不能触发武器的神圣、公正、破敌等附魔的伤害，但是可以触发正义烙印。A类附伤是可以重击翻倍的，由此会附带焰/霜/电/酸爆的额外元素伤害。
  3. 常见B1类（不可以被重击翻倍）附伤有：神圣、公正、（高等）破敌、<破晓花吻（善良版）>的2d6神力伤害，阿罗登剑鞘1d4、焰/霜/电/酸爆（这几个千万别组合在一起，不然重击了就卡死——附伤流BUG会导致无限触发元素狂潮）、战斗祭祀的【守序】/【善良】祝福。无拘乐器的神力伤害，天使道途的各种神力武器附伤，巫妖之力的各种污邪附伤和负能量掌握，偷袭附带1d6钝击伤害的<突袭护腕>。
  4. 常见B2类（可以被重击翻倍）附伤有：正义烙印/辟邪斩/辟乱斩、常备不怠、近程射击、神话近程射击
  5. C类附伤：引领打击和元素狂潮，C类附伤基本上就只能附带焰/霜/电/酸爆的重击额外元素伤害了，如果是被A类附伤触发的，那么A类附伤重击时，C类附伤也会重击（附带焰/霜/电/酸爆的额外元素伤害）。如果是被武器攻击的那次伤害触发的话，就不会被重击翻倍。
  6. 如果用火焰烙印且不存读档，那么火焰烙印将直接触发引领打击（但是有时候不行）

远程

  1. 战斗祭祀的【流水】/【火焰】祝福、杀手的【猎手突袭】、第四章贼裔卖的<部落斥候革甲>、第四章奴隶市场商人卖的<快速应敌护符>、法师7环‘火焰烙印’（存读档）和传旨者的【神之敌】（需要阵营差距2阶）、灵使道途炼金术士卖的<精准手套>等A类附伤，会触发其他的B类附伤（正义烙印、近程射击+神话近程射击、常备不怠、神圣、破敌、公正等---详见近战）。
  2. 远程武器不太好附魔破敌（除了审判官和魔战士以外），所以平时常驻的B类附伤就是近程射击+神话近程射击、常备不怠（点出趁隙射击之后的借机攻击）、神圣（第三章的神圣追击短弓）/公正（第四章的神射手，或者第三章笛拉梅尔的弓）。这些效果在一次攻击会被A类附伤触发多次，实现远程伤害的质变。
